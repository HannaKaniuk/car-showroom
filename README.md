# Car Showroom

Віртуальний автосалон — SPA на React з каталогом автомобілів та відгуками.

## Запуск

```bash
npm install
npm run dev
```

Відкрийте http://localhost:5173

## Збірка

```bash
npm run build
npm run preview
```

## Функціонал

- **/** — каталог автомобілів (DummyJSON API, категорія `vehicle`)
- **/vehicles/:id** — детальна сторінка з відгуками
- Пошук за назвою/брендом/описом, фільтр за брендом, сортування (з валідацією довжини запиту)
- Додавання коментарів з валідацією (обов'язкові поля, max length)
- Локальні коментарі зберігаються в `localStorage` і залишаються після перезавантаження

## Стек

- React 19 + TypeScript
- Vite
- React Router
- Чистий CSS (без UI-фреймворків), responsive 420–1440px
